Bootstrap Components Used

Layout and Structure
  • Container - Responsive fixed-width containers for content sections
  • Grid System (Row/Col) - 12-column responsive layout throughout
  • Flexbox Utilities - d-flex, justify-content-between, align-items-center, gap classes
  • Spacing Utilities - Margin and padding classes (mb-4, py-5, g-4, etc.)
Navigation
  • Navbar - Fixed-top responsive navigation with brand logo
  • Navbar Collapse - Mobile hamburger menu with collapsible navigation
  • Navbar Toggler - Toggle button for mobile navigation
  • Nav Pills/Links - Navigation link styling
  • Scrollspy - Automatic active state updates on nav links based on scroll position
Components
  • Modal - Contact information modal dialog triggered by button
  • Collapse - Expandable module grade tables in Education section
  • Buttons - Various button styles (btn, btn-outline, btn-sm)
  • Badges - Year badges, status indicators, tech stack labels
  • Cards - Project cards, hobby cards, panel layouts
  • Tables - Striped tables for module grades
  • Forms - Contact form with form-control, form-label classes
  • List Group - Quick Facts panel with list-group-flush styling
Utilities and Helpers
  • Display Utilities - d-none, d-sm-inline, d-flex for responsive visibility
  • Text Utilities - text-center, text-muted, fw-bold, text-uppercase
  • Border Utilities - border, rounded-circle, border-0
  • Position Utilities - position-relative, position-absolute, fixed-top
  • Responsive Breakpoints - sm, md, lg, xl breakpoint classes throughout

Custom JavaScript Features

Dynamic features implemented with vanilla JavaScript, separate from Bootstrap functionality:

Theme and Appearance
  • Dark/Light Theme Toggle - Persistent theme switching with localStorage
  • Scroll Progress Bar - Animated progress indicator showing page scroll position
  • Parallax Hero Background - Background image moves at different rate during scroll
Animations and Effects
  • Typewriter Effect - Animated text typing in hero section with multiple phrases
  • Animated Counters - Experience years count up when scrolled into view
  • Card Tilt Effect - 3D perspective tilt on mouse hover for all cards
  • Staggered Card Animations - Cards animate in sequence when section becomes visible
  • Section Fade-in - Sections animate into view using Intersection Observer
  • Skill Bar Animations - Progress bars animate to their values when visible
  • Skill Tooltips - Hover tooltips showing percentage values
Navigation and UX
  • Smooth Scroll Navigation - Smooth scrolling to sections on nav link click
  • Navigation Sliding Indicator - Animated underline that follows active nav item
  • Back to Top Button - Appears after scrolling, smooth scrolls to top
  • Active Section Highlighting - Nav links update based on current scroll position
  • Navbar Scroll State - Navbar background changes when scrolled
  • Keyboard Navigation - Number keys 0-7 for quick section jumping
Interactive Features
  • Project Filter System - Filter projects by year with animated transitions
  • Dynamic Project Modals - Modal content populated from JavaScript data object
  • Contact Form Validation - Client-side form validation with toast notifications
  • Copy to Clipboard - Click to copy email/phone with visual feedback
  • Toast Notifications - Custom notification system for user feedback

Custom CSS Features

  • CSS Custom Properties - Theme variables for colors, spacing, transitions
  • Dark/Light Theme System - Complete dual-theme implementation
  • Glassmorphism Effects - Backdrop blur on navbar and modals
  • Custom Card System - Unified .card base class with .card-accent modifier
  • Gradient Overlays - Hero section and hobby card image overlays
  • Custom Animations - Keyframe animations for bounce, blink, slide effects
  • Responsive Design - Mobile-first with custom breakpoint adjustments
  • Reduced Motion Support - Respects prefers-reduced-motion preference
